# 🎁 Amigo Secreto

Uma aplicação simples do desafio da ONE em **HTML, CSS e JavaScript** para adicionar amigos e sortear um amigo secreto.

---

## 🔹 Funcionalidades

- Adicionar nomes de amigos
- Visualizar a lista
- Sortear aleatoriamente
- Alertas para entradas vazias

---

## 💻 Tecnologias

- HTML5
- CSS3
- JavaScript
